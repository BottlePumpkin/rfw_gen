# rfw_gen v0.2 Design Spec — Core 위젯 전체 확장

## Overview

rfw_gen v0.1에서 구축한 파이프라인(Dart → AST → IR → rfwtxt → .rfw binary) 위에
RFW가 지원하는 **core 위젯 전체**를 매핑한다. GestureDetector(핸들러 필요)는 v0.4로 defer.

## Scope

### 추가 위젯 (32개)

**Layout (13)**:
Align, AspectRatio, Expanded, Flexible, FittedBox, FractionallySizedBox,
IntrinsicHeight, IntrinsicWidth, SizedBoxExpand, SizedBoxShrink, Spacer, Stack, Wrap

**Scrolling (4)**:
GridView, ListBody, ListView, SingleChildScrollView

**Styling (10)**:
ClipRRect, ColoredBox, DefaultTextStyle, Directionality, Icon, IconTheme, Image, Opacity, Padding, Placeholder

**Transform (3)**:
Positioned, Rotation, Scale

**Other (2)**:
AnimationDefaults, SafeArea

### 제외

- GestureDetector — 이벤트 핸들러(`onTap`, `set state.x = value`)는 새로운 아키텍처 개념. Material 위젯(ElevatedButton, ListTile 등)도 핸들러가 필요하므로 v0.4에서 한번에 설계.

## 1. ExpressionConverter 확장

### 현재 구조

ExpressionConverter는 Dart AST Expression을 IrValue로 변환한다.
분기 기준은 AST 노드 타입이며, `ParamMapping.transformer` 키는 현재 사용되지 않는다.

- `MethodInvocation` (target 없음): `Color(0xFF...)` → IrIntValue
- `MethodInvocation` (target 있음): `EdgeInsets.all(8)` → IrListValue
- `PrefixedIdentifier`: `MainAxisAlignment.center` → IrEnumValue
- 리터럴: string, int, double, bool, list

### 추가 변환기

| Dart 입력 | AST 타입 | 변환 로직 | rfwtxt 출력 |
|-----------|----------|-----------|-------------|
| `BorderRadius.circular(8)` | MethodInvocation (target: `BorderRadius`) | `circular` → `[{x: r}]`, `only` → `[tl, tr, bl, br]` | IrListValue of IrMapValue |
| `Duration(milliseconds: 300)` | MethodInvocation (target: null, name: `Duration`) | named arg `milliseconds` 추출 | IrIntValue(300) |
| `Curves.easeIn` | PrefixedIdentifier (prefix: `Curves`) | `Curves` prefix → string 변환 | IrStringValue("easeIn") |
| `RfwIcon.home` | PrefixedIdentifier (prefix: `RfwIcon`) | `RfwIcon` prefix → codepoint lookup → map | IrMapValue({icon, fontFamily}) |
| `NetworkImage('url')` | MethodInvocation (target: null) | URL 추출 → map | IrMapValue({source, scale}) |
| `AssetImage('path')` | MethodInvocation (target: null) | path 추출 → map | IrMapValue({source, scale}) |
| `SliverGridDelegateWithFixedCrossAxisCount(...)` | MethodInvocation (target: null) | named args → map | IrMapValue |

### Icon codepoint 문제

`Icons.home`의 codepoint는 Flutter SDK 런타임 상수로, package:analyzer 빌드타임에 조회 불가.

**해결**: `RfwIcon` 헬퍼 클래스 제공.

```dart
// packages/rfw_gen/lib/src/rfw_icons.dart
class RfwIcon {
  static const int home = 0xe318;
  static const int search = 0xe567;
  // ~50 commonly used icons
}
```

사용자 코드:
```dart
Icon(icon: RfwIcon.home, size: 24)
```

ExpressionConverter에서 `RfwIcon` prefix 인식 → codepoint 조회 → `IrMapValue({icon: IrIntValue, fontFamily: IrStringValue})` 반환.

### _convertMethodInvocation 확장

```dart
IrValue _convertMethodInvocation(MethodInvocation expr) {
  final target = expr.target;
  final methodName = expr.methodName.name;

  if (target == null && methodName == 'Color') return _convertColor(expr);
  if (target == null && methodName == 'Duration') return _convertDuration(expr);
  if (target == null && methodName == 'NetworkImage') return _convertNetworkImage(expr);
  if (target == null && methodName == 'AssetImage') return _convertAssetImage(expr);
  if (target == null && _knownGridDelegates.contains(methodName)) return _convertGridDelegate(expr);

  if (target is SimpleIdentifier) {
    if (target.name == 'EdgeInsets') return _convertEdgeInsets(methodName, expr.argumentList);
    if (target.name == 'BorderRadius') return _convertBorderRadius(methodName, expr.argumentList);
  }

  throw UnsupportedExpressionError(...);
}
```

`_knownGridDelegates`:
```dart
static const _knownGridDelegates = <String>{
  'SliverGridDelegateWithFixedCrossAxisCount',
  'SliverGridDelegateWithMaxCrossAxisExtent',
};
```

### _convertPrefixedIdentifier 확장

```dart
IrValue _convertPrefixedIdentifier(PrefixedIdentifier expr) {
  final prefix = expr.prefix.name;
  final identifier = expr.identifier.name;

  if (_knownEnumPrefixes.contains(prefix)) return IrEnumValue(identifier);
  if (prefix == 'Curves') return IrStringValue(identifier);
  if (prefix == 'RfwIcon') return _convertRfwIcon(identifier);

  throw UnsupportedExpressionError(...);
}
```

### _knownEnumPrefixes 확장

기존에 없는 enum prefix 추가:
- `ImageRepeat` (Image의 repeat 파라미터)

## 2. WidgetRegistry 확장

### 구조 변경

29개 위젯으로 커지므로 `WidgetRegistry.core()` 내부를 카테고리별 static 메서드로 분리:

```dart
factory WidgetRegistry.core() {
  return WidgetRegistry._fromMap({
    ..._textWidgets(),
    ..._layoutWidgets(),
    ..._stylingWidgets(),
    ..._scrollingWidgets(),
    ..._transformWidgets(),
    ..._otherWidgets(),
  });
}
```

### 위젯 매핑 상세

#### Layout (기존 Column, Row, Center + 신규 13개)

**Align**: optionalChild, params: alignment(alignment), widthFactor, heightFactor, duration(duration), curve(curve)

**AspectRatio**: optionalChild, params: aspectRatio

**Expanded**: child, params: flex

**Flexible**: child, params: flex, fit(enum)

**FittedBox**: optionalChild, params: fit(enum), alignment(alignment), clipBehavior(enum)

**FractionallySizedBox**: child, params: alignment(alignment), widthFactor, heightFactor

**IntrinsicHeight**: optionalChild, params: (없음)

**IntrinsicWidth**: optionalChild, params: width, height

**SizedBoxExpand**: optionalChild, params: (없음) — rfwName: `core.SizedBoxExpand`

**SizedBoxShrink**: optionalChild, params: (없음) — rfwName: `core.SizedBoxShrink`

**Spacer**: none, params: flex

**Stack**: childList(children), params: alignment(alignment), textDirection(enum), fit(enum), clipBehavior(enum)

**Wrap**: childList(children), params: direction(enum), alignment(enum), spacing, runAlignment(enum), runSpacing, crossAxisAlignment(enum), textDirection(enum), verticalDirection(enum), clipBehavior(enum)

#### Scrolling (신규 4개)

**ListView**: childList(children), params: scrollDirection(enum), reverse, primary, shrinkWrap, padding(edgeInsets), itemExtent, cacheExtent, clipBehavior(enum)

**GridView**: childList(children), params: scrollDirection(enum), reverse, primary, shrinkWrap, padding(edgeInsets), gridDelegate(gridDelegate), cacheExtent, clipBehavior(enum)

**SingleChildScrollView**: optionalChild, params: scrollDirection(enum), reverse, padding(edgeInsets), primary, clipBehavior(enum)

**ListBody**: childList(children), params: mainAxis(enum), reverse

#### Styling (기존 Container, SizedBox + 신규 10개)

**Padding**: optionalChild, params: padding(edgeInsets), duration(duration), curve(curve)

**ClipRRect**: optionalChild, params: borderRadius(borderRadius), clipBehavior(enum)

**ColoredBox**: optionalChild, params: color(color)

**DefaultTextStyle**: child, params: style(textStyle), textAlign(enum), softWrap, overflow(enum), maxLines, duration(duration), curve(curve)

**Directionality**: child, params: textDirection(enum)

**Icon**: none, params: icon(iconData), size, color(color), semanticLabel

**IconTheme**: child, params: iconThemeData(iconThemeData)

**Image**: optionalChild, params: source(imageProvider), semanticLabel, width, height, color(color), fit(enum), alignment(alignment), repeat(enum)

**Opacity**: optionalChild, params: opacity, duration(duration), curve(curve)

**Placeholder**: none, params: color(color), strokeWidth, placeholderWidth, placeholderHeight

#### Transform (신규 3개)

**Positioned**: child, params: start, top, end, bottom, width, height, duration(duration), curve(curve)

**Rotation**: optionalChild, params: turns, alignment(alignment), duration(duration), curve(curve)

**Scale**: optionalChild, params: scale, alignment(alignment), duration(duration), curve(curve)

#### Other (신규 2개)

**AnimationDefaults**: child, params: duration(duration), curve(curve)

**SafeArea**: child, params: left, top, right, bottom, minimum(edgeInsets)

## 3. RfwIcon 헬퍼

### 파일: `packages/rfw_gen/lib/src/rfw_icons.dart`

자주 사용하는 Material Icons ~50개의 codepoint를 const로 제공:

```dart
class RfwIcon {
  RfwIcon._();

  // Navigation
  static const int home = 0xe318;
  static const int menu = 0xe3dc;
  static const int arrowBack = 0xe092;
  static const int arrowForward = 0xe093;
  static const int close = 0xe16a;

  // Action
  static const int search = 0xe567;
  static const int settings = 0xe57f;
  static const int delete = 0xe1b9;
  static const int add = 0xe047;
  static const int edit = 0xe22b;
  static const int check = 0xe156;
  static const int refresh = 0xe514;

  // Content
  static const int favorite = 0xe25b;
  static const int share = 0xe580;
  static const int send = 0xe571;
  static const int star = 0xe5f9;

  // Communication
  static const int email = 0xe22a;
  static const int phone = 0xe4a2;
  static const int chat = 0xe15b;
  static const int notifications = 0xe42f;

  // ... etc (~50 total)

  /// Lookup map for ExpressionConverter
  static const Map<String, int> _codepoints = {
    'home': home,
    'search': search,
    // ...
  };

  static int? lookup(String name) => _codepoints[name];
}
```

`package:rfw_gen/rfw_gen.dart`에서 export.

ExpressionConverter에서 `RfwIcon.xxx` 인식 시:
```dart
IrValue _convertRfwIcon(String name) {
  final codepoint = RfwIcon.lookup(name);
  if (codepoint == null) throw UnsupportedExpressionError('Unknown RfwIcon: $name');
  return IrMapValue({
    'icon': IrIntValue(codepoint),
    'fontFamily': IrStringValue('MaterialIcons'),
  });
}
```

## 4. 예제 앱 확장

### 신규 예제 위젯 (5개)

`example/lib/widgets.dart`에 추가:

**scrollableList**: ListView + Padding + Icon + Expanded
```dart
@RfwWidget('scrollableList')
Widget buildScrollableList() {
  return ListView(
    padding: EdgeInsets.all(16),
    children: [
      Padding(padding: EdgeInsets.symmetric(vertical: 8), child: ...),
      // Icon + Text rows
    ],
  );
}
```

**imageGallery**: GridView + ClipRRect + Image + AspectRatio

**overlayLayout**: Stack + Positioned + Opacity + Scale

**responsiveWrap**: Wrap + Spacer + Flexible + SafeArea

**animatedCard**: AnimationDefaults + Rotation + ColoredBox + DefaultTextStyle
